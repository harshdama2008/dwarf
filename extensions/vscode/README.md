## Agent

Agent works on development tasks together with your chosen model, using tools to read, edit, and run commands in your workspace.

## Chat

Chat lets you ask general questions and clarify code sections without leaving the editor.

## Edit

Edit modifies a code section in place without leaving your current file.

## Autocomplete

Autocomplete gives you inline code suggestions as you type, powered by a local or remote model of your choice.
